# Commands Reference

Quick reference for useful commands learned through Bandit challenges.

## File Operations
```bash
ls                    # List directory contents
ls -a                 # List all files (including hidden)
ls -la                # List all files with details
cat filename          # Display file contents
file filename         # Identify file type
pwd                   # Print working directory
cd directory          # Change directory
```

## Handling Special Filenames
```bash
cat ./-               # File starting with dash
cat "file name"       # File with spaces
cat ./...hidden       # Hidden file
```

## SSH & Remote Access
```bash
ssh user@host         # Connect to remote server
ssh -p PORT user@host # Connect with specific port
exit                  # Disconnect from SSH session
```

## Terminal Management
```bash
reset                 # Reset terminal after corruption
clear                 # Clear terminal screen
Ctrl + C              # Stop current command
Ctrl + D              # Exit/logout
```

## Getting Help
```bash
man command           # Read manual page for command
command --help        # Quick help for command
```
